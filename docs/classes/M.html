Solves a linear system Ax=b where A is positive definite, using cholesky factorization. Note that the matrix A will be overwritten by the triangular matrix factor, and that the right hand side b will be overwritten by the solution.

Defined in src/tridiag.js:3

Tridiagonal matrix system solver. Solves A*x = v, where A is a tridiagonal matrix. See https://en.wikipedia.org/wiki/Tridiagonalmatrixalgorithm#C

Parameters:

  • v M

    The right hand side. Note that this vector will be overwritten by the solution x.

  • N Number

    The size of the matrix A.

  • a M

    Subdiagonal (means it is the diagonal below the main diagonal)

  • b M

    Main diagonal

  • c M

    Superdiagonal (means it is the diagonal above the main diagonal)
